Off-policy evaluation for slate recommendation

Proposal

Background

We are interested in the evaluation problem in recommendation of an ordered set of items/ ranking based on some user context. This is a common scenario in the domain of web-services search engine, ads displaying and recommendation systems. The ranking in information retrieval requires selection of not only one but multiple actions at the same time. Once the agent chooses the multiple actions, a collective reward characterizing the quality of the entire selection would be observed. The rewards of the individual actions are not necessarily observed. (这是和平时我们做的recommendation system不同的地方，以前每篇文章都有自己的reward)

The entire selection is called slate, and a common method to solve this problem is called combinatorial/slate bandits, which is similar to the traditional multi-armed bandit problem, except that the agent selects a slate consisting of multiple slots, each one of which is occupied by an action.We are planning to study the off-policy evaluation with slate bandits in counterfactual learning, which we think should be very useful in real-world industry settings. Specifically, by off-policy evaluation we mean to use the data collected during operation, i.e. logging policies, to test new policies, i.e. target policies, along evaluation metrics.

Goal

  • Minimum goal:
    • Reproduce the results in paper [1].
    • Analyze the strength and weakness of the evaluation method in [1].
  • Likely goal:
    • Reproduce the results in paper [2].
    • Try different pairs of logging and target policies to examine the proposed estimators.
  • Improvement goal:
    • Try different datasets for semi-synthetic evaluation.
    • Try to find the best policy by optimizing the rewards [3].
