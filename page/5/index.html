Summary

今天刷了union find类型的题目，一共是4道题目:

  • graph valid tree
  • number of connected components
  • number of islands ii
  • friend circle

其实union find就是更加规范化的一种图相关的算法，通常这些题目也可以用dfs或者bfs来解决。之后等在刷dfs和bfs回来整理union-find看看哪些更加适合union find。
